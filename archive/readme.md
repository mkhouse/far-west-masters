# Archive

Historical Far West Masters content kept for reference, organized by season.

## Structure

- `25-26/` — files from the 2025-26 season
  - `email/` — all transactional and announcement emails sent that season
  - `results/` — end-of-season results table snapshots
  - `schedule/` — final season schedule table
- `forerunner/` — full Forerunner newsletter archive, organized by season
  (e.g. `forerunner/2025-2026/2026-01-02.html`)

## Adding to the archive

When you want to back up content for future reference:

1. Take a screenshot and add it to `screenshots/`
2. Create a new file here under the relevant season folder and paste the content in
3. Add a short note describing the process or context if it isn't self-explanatory
